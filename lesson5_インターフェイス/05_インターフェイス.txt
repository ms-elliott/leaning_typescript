[068. 構造的部分型　structual subtyping]
構造的
    同じ構造をした型であれば区別はしない
    例: type Human = { name: string, age:number }
        type Person = { name: string, age:number }
部分型
    取れる値の範囲がより広い型への代入は可能
        string | number にstring を代入する
